This page explains how to use the `@krono/core` SDK.
The `@krono/core` SDK makes it easier to connect to the Kraken Websocket API and store / retrieve historical data for time traveling.
You can find the `@krono/core` SDK source code on Github.
To get started, install the required packages.

<CodeGroup>

    ```pnpm pnpm
    pnpm add @krono/core
    ```

    ```npm npm
    npm install @krono/core
    ```

    ```yarn yarn
    yarn add @krono/core
    ```

    ```bun bun
    bun add @krono/core
    ```
</CodeGroup>

## Initialize

Create and export a new Orderbook instance.

```typescript create-config.ts
import { Orderbook } from '@krono/hooks'

export const orderbook = new Orderbook({
    symbol: "BTC/USD",
})
```

In this example, the Krono Orderbook is configured to use the BTC/USD pair. Check out the config docs for more configuration options.
