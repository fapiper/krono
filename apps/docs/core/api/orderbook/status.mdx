---
title: 'Status'
description: 'Property for the current connection state.'
---

## Import

```ts
import { Orderbook } from '@krono/core'
```

## Usage

Read the current raw status string.

```ts
if (orderbook.status === 'connected') {
  console.log('We are live!');
}
```

## Type

```ts
type ConnectionStatus = 'disconnected' | 'connecting' | 'connected' | 'error'
```
