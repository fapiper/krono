---
title: 'Reconnect'
description: 'Getter and setter for reconnection strategy.'
---

## Import

```ts
import { Orderbook } from '@krono/core'
```

## Usage

Configures how the orderbook behaves when the connection drops.

```ts
orderbook.reconnect = {
  enabled: true,
  maxAttempts: 5,
  delayMs: 2000
}
```

## Type

```ts
{
  enabled: boolean;
  maxAttempts: number;
  delayMs: number;
}
```
