---
title: 'Error'
description: 'Property containing the last error object.'
---

## Import

```ts
import { Orderbook } from '@krono/core'
```

## Usage

Holds the last error if the status is `error`.

```ts
if (orderbook.error) {
  console.error(orderbook.error.message);
}
```

## Type

```ts
Error | null
```
